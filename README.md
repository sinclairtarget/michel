# Michel
<img width="768" height="512" alt="michel" src="https://github.com/user-attachments/assets/9439d100-d93c-4436-8a03-d94da2995e8d" />


Michel is a minimal static site builder based on [MyST Markdown](https://mystmd.org/).

## Model
`content` directory. Files with YAML frontmatter, MyST markdown body.

`site` directory. Templated HTML pages. Every page in the website corresponds
to a templated HTML page. No auto-generation or magic.
